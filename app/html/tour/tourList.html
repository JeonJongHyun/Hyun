        셀하에서 직접 픽업차량이 호텔까지 픽업을 가며, 가이드에게 받은 바우처를 제시하시면 됩니다.
- 불포함사항 : 사진, 비디오, 돌고래와 수영, 옵션비용, 가이드 팁
- 준비물 : 수영복, 편한복장, 모자, 선글라스, 선크림(환경보호용품 선크림만 허용됩니다), 카메라, 수건, 여분의 돈

- 불포함사항 : ATV, 짚라인은 희망고객만 현지에서 유료결제, 현지인 운전기사 팁(봉사료) $10/인 별도 지불

우리만의 칸쿤을 사진으로 남기자!  1 커플 1시간 촬영 (최대 2커플 촬영)

- 장소 : 호텔존 내 퍼블릭 비치
- 이동 : 촬영시간 10분전까지 개별이동하여 한국인 촬영팀 미팅
- 원본 70장 + 보정본 15장. 이메일로 전송 (촬영 후 2달 소요)
- 촬영 시간은 오전 10시 / 11시 중 택 1
- 불포함 사항 : 사진 기사 팁 $20은 현지 지불 (1인 당), 의상/소품/메이크업

   [선택 4.] 칸쿤 정글투어
정글투어는 칸쿤을 대표하는 투어중에 하나로 직접 운전하는 2-3인승 모터보트를 타고

- 시간 : 오전 10시 / 12시 / 3시 타임 중 택 1(소요시간은 2~3시간 입니다)
